# MLLMChat

基于 **React + Next.js Pages Router + Tailwind CSS + Vercel AI SDK** 的多模态聊天应用。

## 功能

- 文本聊天：DeepSeek V4 Pro
- 图片生成：doubao-seedream-4.0
- 视频生成：Seedance 1.5 Pro
- 统一 API：`/api/ai-sdk/multimodal`
- 简洁风格页面与流式响应体验

## 快速开始

1. 安装依赖

```bash
npm install
```

2. 配置环境变量

```bash
cp env.example .env.local
```

3. 启动项目

```bash
npm run dev
```
